# System Tray in C3
![img](screenshot.png)

### tray.c3
Create a system [tray](tray.c3) status bar with a menu

> c3c run tray

### window.c3
Example usage of the Objective-C Runtime through C3's objc bindings. 

An empty [window](window.c3) with events

> c3c run window
